# LipuPonaMi
This app lists useful websites for research activities. The name "lipu pona mi" means "my good websites" in Toki Pona.

## How to Use
Run the following command:
```bash
python3 run_app.py
```
Then, access [http://localhost:8282/](http://localhost:8282/).
